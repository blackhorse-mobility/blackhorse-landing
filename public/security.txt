# Security Policy

This file is for coordinating the disclosure of security vulnerabilities found in https://black-horse.io.

If you discover a security vulnerability, please email security@black-horse.io instead of using public issue trackers.

We will respond to your report as soon as possible and work with you to resolve the issue.
